# FastAPI Blog

Fastapi tutorial blog application.

https://www.youtube.com/watch?v=7t2alSnE2-I&t=13861s

## Installation

1. Install poetry

https://cocoatomo.github.io/poetry-ja/

2. Install dependencies

```
$ poetry install
```

## Basic usage

Run api server.

```
$ uvicorn blog.main:app --reload
```

Open API docs.

http://127.0.0.1:8000/docs
